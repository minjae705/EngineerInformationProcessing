79. 다음 SQL문장의 실행 결과에 들어갈 알맞은 용어를 쓰시오.

[성적 테이블]
학번 과목    성적
1000 보안     A
1001 운영체제 B
1002 통계     A+
1003 암호학   B+
1005 알고리즘 A
1006 보안     A+

[SQL문]
select 과목 
from 
( 
select 과목, count(과목) 
from 성적 
group by 과목 
having count(과목) > 1 
) a;

[실행결과]
과목  
(   )

답) 보안

과목 count(과목)
보안  2

group by 를 이용하여 과목별로 건수(count)를 집계한 후, 과목의 건수가 1보다 큰 데이터만 추출
