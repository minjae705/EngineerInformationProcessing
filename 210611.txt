78. 다음은 디지털 저작권 관리(DRM) 구성요소에 대한 설명이다. 괄호 안에 들어갈 가장 올바른 용어를 쓰시오.

- 콘텐츠 제공자(Contents Provider) : 콘텐츠를 제공하는 저작권자
- (  1  ) : 콘텐츠를 메타 데이터와 함께 배포 가능한 단위로 묶는 기능
- (  2  ) : 배포된 콘텐츠의 이용 권한을 통제
- (  3  ) : 원본을 안전하게 유통하기 위한 전자적 보안 장치

정답) 
(1) 패키저(Packager)
(2) DRM 컨트롤러(DRM Controller)
(3) 보안 컨테이너(Security Container)
