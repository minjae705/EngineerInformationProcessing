63. 다음 C 프로그램 결과를 쓰시오. 

#include <stdio.h>

void main(){
	char A = 'A';  // 65
	char a = 'a';   //97
	printf("%x\n", a%A);
}

답: 20

해설) 아스키(ASCII)코드에서 A는 65, a는 97이다.
97%65인 32가 된다.
%x 로 출력을 하면 십진수 32가 16진수인 20으로 출력 된다.
